# Getting started
## Hello elephant!
```
slon('Hello elephant!')
```

## Variable assignment
```
a @= 10
b @= "fuck it"
```

## Changing variable value
```
a = 5
b = "fuck it x2"
```

## "if" expression
 Supports operands: `<` `>` `!=` `==`
```
if b != "fuck" {
    ...
}
```
